
### Option 3: Manual PDF Generation
1. Open `portfolio/portfolio-pdf.html` in your browser
2. Press `Cmd+P` (Mac) or `Ctrl+P` (Windows/Linux)
3. Select "Save as PDF"
4. Save as `autonomous-redteam-portfolio.pdf`
